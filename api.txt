进房欢迎和送礼都是系统群消息
消息内容是个json
{
    Type int
    Msg string
}
Type=0表示文字消息 Msg直接显示在文字区域即可
Type=1表示送礼滚屏消息 Msg是个json 格式如下
{
    SendID int
    SendName string
    ReceiveID int
    ReceiveName string
    GiftID int
}
Type=2表示房间在线列表消息 Msg是个json 格式如下
{
    RoomID int
    OnlineUserCnt int
    OnlineUserList [
        {
            UserId int
            AvatarFile string
        },
        ...
    ]
}
Type=3表示房间数据更新消息 Msg是个json 格式如下
{
    RoomID int
    GiftValue int（房间收礼总额）
    LikeNum int（点赞数）
}
